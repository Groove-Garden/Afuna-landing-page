גינה של גרוב - דף נחיתה אפונה

מה עודכן בחבילה הזו:
1. הוחלף הלוגו לתמונה החדשה שסופקה.
2. הוטמע טופס Smoove שעובד ומצרף נרשמים לרשימה.
3. הטמעת הטופס סודרה בתוך אזור מוגדר כדי שלא תשבור את מבנה הדף.
4. הטופס התחתון הוחלף בכפתור שמוביל לטופס הראשי.
5. נשמרה התאמה למובייל באמצעות ה-CSS המקורי ותוספות רספונסיביות.

להעלאה לאתר:
החליפו את הקבצים הקיימים ב-GitHub בקבצים שבתיקייה הזו, בעיקר index.html ותיקיית assets.
